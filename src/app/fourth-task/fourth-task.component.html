Less is a pre-processor styling language.

Preprocessors such as less or scss allow us (programmers) to use dynamic features such as variables, functions, loops, imports and more in our stylesheet to create a more elegant and maintainable style for our front end application.
Such preprocessors can then be compiled to standard CSS for our runtime environment.
For a detailed brief on styling preprocessors, check the documentation for less or sass
